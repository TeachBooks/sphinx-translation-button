# Example book

This is an example book which uses sphinx internationalization (https://www.sphinx-doc.org/en/master/usage/advanced/intl.html).
For the button to work the following structure is expected

- _build
  - en
     - english book
  - nl
     - dutch book

The required languages will also need to be specified in the _config.yml file. Look at the release-package for more info.
